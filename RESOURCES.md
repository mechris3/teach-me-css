# CSS Mastery Resources

## Knowledge

- [MDN: Custom properties (--*)](https://developer.mozilla.org/en-US/docs/Web/CSS/--*/)
  Canonical reference for CSS custom properties. Use for: syntax, scoping, cascade behaviour, computed values.

- [MDN: @property at-rule](https://developer.mozilla.org/en-US/docs/Web/CSS/Guides/Properties_and_values_API/Registering_properties)
  Registered custom properties via Houdini. Use for: type checking, inheritance control, initial values, animating custom properties.

- [MDN: @function at-rule](https://developer.mozilla.org/en-US/docs/Web/CSS/@function)
  CSS custom functions (experimental, Chrome 139+). Use for: reusable parametrised logic in pure CSS.

- [MDN: CSS custom functions and mixins guide](https://developer.mozilla.org/en-us/docs/Web/CSS/Guides/Custom_functions_and_mixins)
  Overview of the Functions and Mixins module. Use for: understanding the full picture of @function and future @mixin.

- [W3C: CSS Functions and Mixins Module Level 1](https://www.w3.org/TR/css-mixins-1/)
  The actual spec. Use for: authoritative syntax, edge cases, formal grammar.

- [Chrome Developers: CSS attr() gets an upgrade](https://developer.chrome.com/blog/advanced-attr)
  Advanced attr() — typed values beyond content property. Use for: using data attributes as CSS values for any property.

- [Una Kravets: New capabilities for attr()](https://una.im/advanced-attr/)
  Practical walkthrough of advanced attr(). Use for: real-world examples, browser support details.

- [Lea Verou: The --var: ; hack to toggle multiple values](https://www.verou.me/blog/2020/10/the-var-space-hack-to-toggle-multiple-values-with-one-custom-property/)
  Space toggle technique. Use for: boolean-like switching with custom properties, conditional styling without JS.

- [Lea Verou: Custom properties with defaults](https://lea.verou.me/blog/tags/css-variables)
  Collection of posts on advanced custom property patterns. Use for: API design for components, fallback strategies.

- [Kevin Powell: Master CSS Custom Properties (Frontend Masters)](https://frontendmasters.com/tutorials/kevin-powell/master-css-custom-properties/)
  Video tutorial on locally-scoped, undeclared, and registered custom properties. Use for: visual explanations, practical demos.

- [Frontend Masters: What You Need to Know about Modern CSS (2025 Edition)](https://frontendmasters.com/blog/what-you-need-to-know-about-modern-css-2025-edition/)
  Chris Coyier's curated overview of modern CSS features. Use for: big-picture orientation on what's new and worth knowing.

- [OddBird: Custom CSS Functions in the Browser](https://www.oddbird.net/2025/04/11/custom-functions/)
  Early practical exploration of @function. Use for: real-world usage patterns, gotchas, progressive enhancement.

- [W3C: CSS Snapshot 2024](https://drafts.csswg.org/css-2024/)
  Official state-of-CSS document. Use for: knowing which specs are stable vs drafts.
